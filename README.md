## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Updating the Favicon

To replace the favicon (the icon shown in browser tabs):

1. Go to [https://favicon.io/favicon-converter/](https://favicon.io/favicon-converter/)
2. Upload your image (PNG or SVG works best)
3. Download the generated package
4. Replace the following files in the `public/` folder with the ones from the download:
   - `favicon.ico`
   - `favicon-16x16.png`
   - `favicon-32x32.png`
   - `apple-touch-icon.png`
   - `android-chrome-192x192.png`
   - `android-chrome-512x512.png`
